# INDEX
#### CREATE SCHEMA xxx;  
- 创建数据库;
- 等同于 CREATE DATABASE xxx;

#### PK、NN、UQ、B、UN、ZF、AI、G 代表含义
- PK: PRIMARY KEY, 主键;
- NN: NOT NULL, 非空;
- UQ: UNIQUE, 唯一索引;
- B: BINARY, 二进制数据;
- UN: UNSIGNED, 无符号整数(非负数);
- ZF: ZERO FILL, 填充0;
- AI: AUTO INCREMENT, 自增;
- G: GENERATED, 生成列。
