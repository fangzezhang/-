# 7.生命周期
- 小程序应用生命周期;
- 小程序页面生命周期;

### 小程序应用生命周期
- onLaunch: 小程序初始化完毕后触发;
- onShow: 小程序切换成前台状态;
- onHide: 小程序切换成后台状态;
- onError: 小程序发生脚本错误或 API 调用失败时;

### 小程序页面生命周期
- onLoad: 页面初次加载时, 页面未被销毁前只调用一次;
- onShow: 页面显示之后或者从其他页面返回当前页时调用;
- onReady: 页面初次渲染完成后, 在 onShow 之后调用, 页面未被销毁前只调用一次;
- onHide: 在当前页面的基础上打开新的页面时触发;
- onUnload: 关闭当前页;
